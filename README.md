# GreyChair Grooming (Demo)

This repo contains everything you need to run the app locally.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Create `.env.local` and set `VITE_GEMINI_API_KEY` to your Gemini API key:
   `VITE_GEMINI_API_KEY=...`
3. Run the app:
   `npm run dev`
